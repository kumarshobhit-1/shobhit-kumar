## ⚡ Highlights & Key Features

- **🎨 Unique Neobrutalist Design System**: Bold ink borders, high-contrast palette, dynamic dark/light mode toggle, and micro-interactions.
- **💻 Interactive Developer Terminal**: Built-in bash-style terminal emulator supporting commands (`bio`, `projects`, `skills`, `certificates`, `experience`, `stats`, `verify <num>`, greetings/farewells, `clear`, and `help`).
- **📱 100% Fully Responsive Layout**: Tailored mobile breakpoint architecture (`@media (max-width: 768px)`) featuring full-screen slide-out drawers, responsive grids, and touch-optimized navigation.
- **🚀 Project Showcase & Live CTAs**: Interactive project polaroid cards with GitHub repository quick-links, live website launch CTAs, and slide-out detail drawers.
- **🎖 Credential Verification Drawer**: Achievements list in descending chronological date order with direct verification links and image preview capabilities.
- **📬 Real-time Web3Forms Contact Drawer**: Functional contact form powered by Web3Forms API delivering instant email notifications without backend server dependencies.
- **📊 Global Visit Telemetry**: Live visitor counter tracking global traffic with StrictMode double-increment prevention.
- **🔍 Custom 404 Error Fallback**: Dedicated 404 Not Found route handler with full navigation bar and section redirection logic.

## 🛠️ Tech Stack & Architecture

- **Frontend Core**: React 19, JavaScript (ES6+), HTML5
- **Styling & Motion**: Custom Vanilla CSS (Tokens, CSS Grid, Flexbox, Glassmorphism, Micro-animations)
- **Icons & Graphics**: Custom SVG doodle graphics & Lucide React Icons
- **Build Tool**: Vite v8.1 (Fast HMR & production bundle optimization)
- **Deployment**: Vercel / Netlify / GitHub Pages

## 💻 Terminal Emulator Commands

| Command | Action |
| :--- | :--- |
| `projects` / `ls` | Lists all 5 production projects dynamically |
| `project <1-5>` | Displays deep-dive details, tech stack, and GitHub links |
| `skills` / `skill` | Displays categorized engineering capabilities |
| `experience` / `exp` | Shows professional SDE experience at TVK |
| `certificates` | Lists all 8 certifications in descending chronological order |
| `verify <1-8>` | Opens credential verification info and external URL |
| `bio` / `cat bio.md` | Displays developer summary and technical focus |
| `stats` | Renders production application telemetry stats |
| `hi` / `hello` / `bye` | Case-insensitive terminal greeting & farewell responses |
| `clear` / `help` | Clears terminal history or displays command index |

## ⚙️ Local Development Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/kumarshobhit-1/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser.

4. **Build for production**:
   ```bash
   npm run build
   ```
